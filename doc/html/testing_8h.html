#define TEST (   test_class,
  test_case 
)    void test_class##_test::test_case()

Declares the test case member function of a test fixture class.

Parameters
test_classThe name of the class under test. This class must have a corresponding test fixture class named as itself plus the "_test" suffix.
test_caseThe name of the test case member function of the fixture class.

A good design convention dictates that every test fixture class in the project must be named with the "_test" suffix. Having to type such suffix in every test case definition reduces readability and increases code duplication. To overcome these problems, this macro may be used to declare a test case in its definition.

Example:

class Foo {};
class Foo_test: public CppUnit::TestFixture
{
CPPUNIT_TEST_SUITE (Foo_test);
CPPUNIT_TEST (test_case);
CPPUNIT_TEST_SUITE_END();
void test_case();
};
CPPUNIT_TEST_SUITE_REGISTRATION (Foo_test);
TEST (Foo, test_case) // Equivalent to: void Foo_test::test_case()
{
// Test code goes here.
}
